EMP MCP Interface

The MCP stdio interface lets AI clients discover modernization packs, compare public evidence, analyze a project, and inspect a dry-run transformation plan without applying changes.

Recommended Workflow

1. Discover packs

Call emp.packs to choose the modernization track.

2. Compare evidence

Call emp.benchmarks to inspect published benchmark proof for that pack.

3. Analyze project

Call emp.analyze for readiness, applicability, findings, and enterprise rules.

4. Plan migration

Call emp.transformPlan to inspect planned changes before execution.

Tools

ToolPurposeInputs
emp.packsList packs or return one pack's full metadata.id optional
emp.benchmarksSummarize published benchmark evidence.pack, source, validationStatus, limit optional
emp.analyzeRun readiness analysis with optional enterprise rules.path, pack, rules
emp.transformPlanCreate a dry-run transformation plan without applying file changes.path, pack, engine, recipe, recipeArtifact
