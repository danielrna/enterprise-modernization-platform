Friction Found

ObservationDocumentation Change
Multi-sample repositories can be run from the wrong root.The quickstart now tells users to run from the actual application folder, such as complete/.
Spring Boot 3.x apps should not get a Spring Boot 2 to 3 readiness score.The quickstart now explains that not applicable is honest evidence and suggests other packs.
Low readiness scores create clearer paid work than green demos.The trial page now separates cleanup, Jakarta migration, validation, and pack-selection offer branches.
Older Java 8 Spring Boot 2.x examples need runtime planning before migration execution.The matrix now exposes Java version and Spring Boot version together for consultant scoping.
Public GitHub examples disappear or move.External proof is based on cloned repositories and checked-in report artifacts, not assumptions.
