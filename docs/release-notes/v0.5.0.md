# v0.5.0

Phase 1 Evidence Depth completion with every public benchmark report backed by a real checkout.

## Phase 1 evidence depth complete

Promote the remaining catalog-only benchmark reports to checkout-backed evidence and preserve honest validation outcomes for heavyweight repositories.

Audience: Consultants and maintainers who need the public proof set to be hard to dismiss

- Raises current evidence to 75 public reports, 75 checkout-backed reports, and 17 passing validation reports.
- Promotes Flowable Engine, Spring Cloud Stream, Spring Cloud Contract, Spring Cloud Netflix, and Spring Cloud Kubernetes to checkout-backed reports.
- Promotes Broadleaf Commerce, OpenMRS Core, Kill Bill, Camunda BPM Platform, and Activiti to checkout-backed reports.
- Promotes Apache Fineract, Netflix Conductor, Spring Cloud Data Flow, Alibaba Nacos, and Apache SkyWalking to checkout-backed reports.
- Promotes Apache ShardingSphere, Baeldung Tutorials, Keycloak, Keycloak Jakarta Readiness, and ThingsBoard to checkout-backed reports.
- Keeps validation evidence honest with captured compile/test failures, timeouts, signal-derived exits, and checkout revisions instead of converting failures into catalog-only examples.
- Extends timeout cleanup so Maven Surefire, Gradle workers, and other checkout-working-directory child processes are terminated after timeout evidence is captured.
